Deployment Options

Two ways to integrate. Pick what fits.

Whether your app already has an AI assistant or you're starting fresh, there's a clean path to integration.

For apps with AI

Tool / Function Calling

Your AI assistant calls our coaching API as a tool. The SonderMind coach runs as a function within your existing conversational flow.

  • Works with OpenAI, Anthropic, or custom LLM stacks
  • Coach runs as a tool call within your agent
  • Full context passing between your AI and ours
  • REST API + WebSocket for real-time sessions
  • Assessment scores and coaching insights as structured data
For apps without AI

Embeddable SDK

Drop in a pre-built coaching interface with a few lines of code. Full UI included, fully themeable to match your brand.

  • React Native and Web components
  • Themeable UI that matches your brand
  • Handles all coaching UX out of the box
  • Event callbacks for assessment scores and session state
  • Works on iOS, Android, and web
